Detailed Description

The pause menu manager.

You can extend this to make your own. Everything is pretty modular, so creating you own based off of this should be easy. Thanks for downloading and good luck!

void GreatArcStudios::PauseManager::lodBias ( float  LoDBias)

Change the lod bias using QualitySettings.lodBias = LoDBias / 2.15f; LoDBias is only divided by 2.15 because the max is set to 10 on the slider, and dividing by 2.15 results in 4.65, our desired max.

However, deleting or changing 2.15 is compeletly fine.
